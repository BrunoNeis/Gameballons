#### About 
GHOSt ARROW is a 2D arcade game develoaped in Pygame. The game has 4  level and each level and eith a time event.
There is necessary finish the 4 level for winn and save your score in top 11.
It can be played in cooperative mode for 2 players.
Score is saved in database(SQLite3)
